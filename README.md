# Task 3 - Backend Integration and API Development

## 📱 Android Internship Project

This project was developed as part of the Android App Development Internship at ApexPlanet Software Pvt. Ltd.

## 🚀 Project Overview

The application demonstrates backend integration in Android using REST APIs and Retrofit. The app fetches real-time joke data from an online API and displays it dynamically in the UI.

## ✨ Features

* REST API Integration
* Real-time data fetching
* Dynamic UI updates
* Backend connectivity using Retrofit
* JSON data parsing
* Interactive button functionality
* Internet permission handling

## 🛠️ Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Retrofit
* Gson Converter
* REST API

## 📂 API Used

Official Joke API:
https://official-joke-api.appspot.com/jokes/random

## 📱 Application Functionality

* Displays a button to fetch jokes
* Connects to online API service
* Retrieves random jokes from backend
* Displays fetched content dynamically on screen
